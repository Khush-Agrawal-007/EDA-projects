How to run locally
1. Create and activate virtual environment, then install dependencies:
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install pandas matplotlib seaborn notebook nbconvert
```
2. Run the notebook or export to HTML:
```powershell
jupyter nbconvert --to html Zomato.ipynb --output Zomato_report.html
```
